# Multimodal Chatbot Project

## Project Overview

This project implements an AI-powered **multimodal chatbot** using Python, Streamlit, and Google Gemini API. Unlike a basic text chatbot, this chatbot can process multiple input types such as **text, images, audio, and documents**, making interactions more dynamic and intelligent.

It allows users to communicate with the chatbot naturally and receive meaningful responses based on the provided input.

## 🎯 Objectives

* To understand how multimodal chatbots work
* To integrate generative AI using API keys
* To process multiple forms of user input (text, image, audio, document)
* To implement error handling in AI systems
* To build an interactive web-based chatbot interface

## ⚙️ Technologies Used

* Python
* Streamlit
* Google Generative AI (Gemini 2.5 Pro)
* dotenv (.env)
* File Handling Libraries

## 🔄 How the Multimodal Chatbot Works

1. User provides input (text, image, audio, or document)
2. The input is sent to the Gemini AI model
3. The AI analyzes the input based on its type
4. The chatbot generates and returns an intelligent response
5. The user can continue the conversation interactively

## ✨ Features

* Supports **text-based conversations**
* Accepts **image inputs** for analysis
* Processes **audio inputs** for understanding
* Handles **document uploads** for content-based queries
* Interactive **Streamlit web interface**
* Secure API key management using `.env`
* Error handling for invalid inputs and API issues
* User-friendly and responsive design
