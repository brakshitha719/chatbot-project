# chatbot-project

Project Overview

This project implements an AI-powered chatbot using Python and a generative AI model. The chatbot can respond to user queries in natural language and demonstrates the basics of conversational AI.

🎯 Objectives

To understand how chatbots work

To integrate generative AI using API keys

To handle user input and responses

To implement error handling in AI systems

🧠 Key Concepts Used

Artificial Intelligence

Generative AI

API integration

Error handling

⚙️ Technologies Used

Python

Google Generative AI (Gemini)

Environment variables (.env)

🔄 How the Chatbot Works

User enters a message

The message is sent to the AI model

The AI processes the prompt

The chatbot returns a response

✨ Features

Interactive conversation

Simple console-based interface

Secure API key usage

Handles invalid inputs and errors
